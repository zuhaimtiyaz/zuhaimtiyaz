## 🚀 Current Project

**FPGA Based Fly by Wire System**

Currently developing the initial prototype using Arduino Uno. The long 
term goal is to implement the complete system on an FPGA with autonomous takeoff and landing on circular runways

Features:
Project

1. Fly by Wire (FBW) Control System
2. Autonomous Flight Control
3. Circular Runway Edge Takeoff and Landing
4. PID Based Flight Stabilization
5. Real Time Sensor Integration
6. Wireless Communication using NRF24L01
7. Auto and Manual Flight Modes
8. Emergency Stop Function
9. Obstacle Detection and Avoidance
10. GPS Based Navigation
11. Arduino Based Prototype
12. FPGA Oriented System Design
13. Modular Architecture
14. Breadboard-Based Implementation
15. Cost Effective Prototype
16. Real Time Data Processing
17. Autonomous Decision Making
18. Scalable and Upgradable Design
19. Low Latency Control System
20. Future FPGA Migration Support
